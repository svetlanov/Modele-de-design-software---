# Лабораторная работа 7  
## Тема: Реализация алгоритмов в языке программирования

В этой лабораторной работе я реализовала алгоритм попарного сложения двух списков (из предыдущей лабы) в виде кода на C#. Целью было научиться переводить подробный пошаговый алгоритм в код, используя массивы, циклы и функции.

---

## Постановка задачи

Нужно реализовать на C# алгоритм попарного сложения элементов двух массивов одинаковой длины:

- на вход подаются два массива одинаковой длины;
- необходимо вычислить сумму соответствующих элементов:  
  `result[i] = a[i] + b[i]` для всех допустимых `i`;
- результат записывается в отдельный выходной массив.

Требования лабораторной:

1. В коде должна быть отдельная функция, реализующая **только логику алгоритма**, без печати в консоль.  
2. Массивы создаются в **основной функции** и передаются в функцию-алгоритм параметрами.  
3. Выходной массив также создаётся в основной функции и передаётся в функцию по ссылке (как параметр).  
4. В консоль печатает только основная функция.  
5. Реализовать **две версии**: с использованием `for` и `while`.  
6. Добавить **контракты**: проверки длин массивов и корректности входных данных (через `ArgumentException`).

## Реализация на C#

```csharp
int[] arr1 = new int[6];
arr1[0] = 3;
arr1[1] = 7;
arr1[2] = 45;
arr1[3] = 8;
arr1[4] = 2;
arr1[5] = 17;

int[] arr2 = new int[6]
{
    1,
    5,
    7,
    43,
    9,
    6,
};

static int[] SumNumbers(int[] arr1, int[] arr2)
{
    int arr1_length;
    int arr2_length;
    arr1_length = arr1.Length;
    arr2_length = arr2.Length;

    if (arr1_length != arr2_length)
    {
        throw new ArgumentException("Списки не равны");
    }

    int[] result = new int[arr1_length];

    // for (int i = 0; i < arr1_length; i++)
    // {
    //     int sum_i;
    //     sum_i = arr1[i] + arr2[i];
    //     result[i] = sum_i;
    // }

    int i = 0;
    int sum_i;

    while (true)
    {
        if (i >= arr1_length)
        {
            break;
        }
        sum_i = arr1[i] + arr2[i];
        result[i] = sum_i;
        i++;
    }

    return result;
}

static void print(int[] arr)
{
    for (int i = 0; i < arr.Length; i++)
    {
        Console.WriteLine(arr[i]);
    }
}

int[] result = SumNumbers(arr1, arr2);
print(result);
```
