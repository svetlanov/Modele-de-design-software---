# Лабораторная работа 3  
## Тема: Настройка git

В этой лабораторной я настроила систему контроля версий git для моего проекта, создала .gitignore и связала локальный репозиторий с GitHub.

---

## 1. Инициализация git-репозитория

Я перешла в папку своего проекта:

```bash
cd MyProject
```

И инициализировала репозиторий:

```bash
git init
```

Появилась скрытая папка `.git` — значит репозиторий создан.

---

## 2. Создание .gitignore

Чтобы временные файлы и артефакты компиляции не попадали в коммиты, я создала файл:

**.gitignore**
```gitignore
bin/
obj/
.vs/
*.user
*.suo
```

Теперь эти файлы git будет игнорировать.

---

## 3. Настройка информации пользователя

Один раз настроила глобальные данные:

```bash
git config --global user.name "Svetlana Novac"
git config --global user.email "svetlana********@gmail.com"
```

---

## 4. Добавление файлов и первый коммит

Проверила статус:

```bash
git status
```

Добавила все файлы проекта:

```bash
git add .
```

Сделала первый коммит:

```bash
git commit -m "Initial commit: проект и .gitignore"
```

---

## 5. Создание GitHub-репозитория

Через сайт GitHub создала репозиторий:

```
Modele-de-design-software
```

Без README, чтобы избежать конфликтов.

---

## 6. Связывание локального репозитория с GitHub

Добавила удалённый репозиторий:

```bash
git remote add origin https://github.com/<логин>/csharp-labs.git
```

Выполнила первый push:

```bash
git push --set-upstream origin main
```

Теперь локальный репозиторий связан с GitHub, и изменения можно отправлять.
